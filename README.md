# Mind-Duel Game
https://mind-duel-bymayank.netlify.app/

Mind Duel is a real-time multiplayer number guessing game where two players secretly pick numbers (1–100) and take turns guessing using “UP” and “LOW” hints.

🔹 Key Features
👥 Multiplayer (cross-device) using game codes
🤖 AI mode for solo play
🎨 Modern UI with dark/light mode
📱 PWA support (install like an app)
⚡ No server needed (PeerJS WebRTC)
🎲 Gameplay
Create or join a game with a code
Both players choose a secret number
Take turns guessing
Get hints: UP (higher) / LOW (lower)
First correct guess wins 🎉
🛠 Tech Stack
HTML, CSS, JavaScript
PeerJS (real-time connection)
PWA features
